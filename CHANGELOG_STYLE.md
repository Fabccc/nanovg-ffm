# Changelog Style Guide

1. **Main Title (H1)**: Must be the version number (e.g., `# 1.0.0`).
2. **Section Title (H2)**: Must be "Changelog".
3. **Categories (H3)**: Use descriptive categories **WITHOUT ANY** emojis (e.g., `### New Features`).
4. **Items**: Use bullet points for individual changes.
5. **Language**: English.
